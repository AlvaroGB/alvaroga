# Math Atlas

A standalone, kid-friendly math learning web app.

## Features
- 7 core math worlds
- adaptive difficulty
- one-question-at-a-time gameplay
- saved browser progress
- deep-dive explanations for stronger learners

## Run locally

Use any static server, for example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
