# Lengua Culta

Prototype web app for Spanish lexical cultivation.

Focus order:
1. Nuevos conceptos
2. Precisión conceptual
3. Registro natural
4. Matices y sinónimos útiles
5. Sinónimos raros y ornamento
6. Arcaísmos y relicarios

The app is intentionally simple and static so it can live as a clean entry on the main domain.
