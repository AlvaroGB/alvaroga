Personal Product — Built With AI, Used By Real People

Hanzi Samurai

Chinese Character Learning Game · hanzi-game.vercel.app

A full game, not a flashcard app. Characters fall from the top of the screen — you type their pinyin before they crash. Three modes (Recognition, Recall, Listening), combo multipliers, streak milestones, and wave-based progression mapped to HSK levels 1–3. Spaced repetition engine (SM-2) under the hood weights which characters appear based on your mastery. Confusables engine surfaces look-alike hanzi you actually mix up. Custom audio pipeline via Web Speech API (zh-CN). Lifecycle funnel guides users from first character to daily habit. Built iteratively with Cursor and autonomous agent swarms.
